🎯 문제 정의 (Problem Definition)

1. 배경 (Background)

외국어 학습에 대한 수요는 꾸준히 증가하고 있으나, 많은 학습자가 실질적인 연습 단계에서 큰 장벽에 부딪힙니다.

2. 핵심 문제 (The Problem)

  • 비용 부담 (High Cost): 일대일 튜터링이나 어학원 수강은 효과적일 수 있으나, 매월 발생하는 비용 부담이 커서 지속하기 어렵습니다.
  • 심리적 장벽 (Psychological Barrier): 많은 학습자가 자신의 미숙한 외국어 실력을 타인에게 노출하는 것을 부끄럽게 생각합니다.
  • 시공간적 제약 (Limited Accessibility): 기존 온라인 학습 툴은 인터넷 연결이 필수적이거나, 학원은 정해진 시간에 방문해야 합니다.

3. 타겟 사용자 (Target Users)

  • 비용 문제로 일대일 교습이나 학원 수강을 망설이는 독학 학습자.
  • 틀리는 것을 두려워하여 사람 앞에서는 연습을 꺼리는 내성적인 학습자.
  • 인터넷이 불안정한 환경에서도 꾸준히 학습하길 원하는 바쁜 직장인/학생.

4. 우리가 해결하려는 것 (Our Solution)

이러한 학습자들을 위해, 언제 어디서나 인터넷 연결 없이(Offline), 타인의 시선을 의식할 필요 없이(Private), 비용 부담 없이(Cost-effective) 사용할 수 있는 AI 외국어 교정 튜터를 제공합니다. 사용자는 완전히 프라이빗한 환경에서 AI를 상대로 마음껏 실수하고 즉각적인 피드백을 받으며 실질적인 외국어 자신감을 키울 수 있습니다.

아키텍처 / 설계

백엔드 On-Device LLM: llama.cpp를 통해 경량화된 모델(Phi-3, Llama 3 8B)을 구동, 대화 생성 및 문법 교정을 수행합니다.

클라이언트: Expo React Native (TypeScript)로 개발된 크로스플랫폼 모바일 앱이 UI 역할을 합니다.

통신: 개인 WiFi 네트워크를 통해서만 로컬 PC에서 실행 중인 백엔드 서버(Flask)와 API (HTTP/WebSocket) 통신을 수행합니다.

음성 처리: On-Device STT (Whisper.cpp) 및 On-Device TTS (Piper) API를 제공하여 음성을 텍스트로 변환하고 텍스트를 음성으로 합성합니다.

시나리오 관리: '시나리오 관리자'가 상황극에 필요한 시스템 프롬프트, AI 페르소나, 대화 흐름 규칙(JSON/YAML)을 관리하고 LLM 파이프라인을 제어합니다.

핵심 기능

1. 온디바이스(On-Device) LLM을 통한 100% 오프라인 교정 및 대화

로컬 PC 백엔드(Python Flask)에서 llama.cpp (혹은 Ollama)를 통해 경량화된 LLM(예: Llama 3 8B, Phi-3)을 직접 구동했습니다. 이를 통해 인터넷 연결이 불가능한 환경에서도 (1)실시간 문법 교정 및 (2)상황극(Role-play) 대화가 가능한 튜터 기능을 구현했습니다. 모든 데이터가 로컬 네트워크 내에서만 처리되므로 API 비용이 없으며 사용자의 민감한 대화 내용 유출을 원천적으로 차단합니다.

2. 실시간 오프라인 음성 처리(STT/TTS)를 통한 몰입형 대화 경험

로컬 백엔드 서버에 Whisper.cpp(음성인식) 및 Piper(음성합성) 모델을 API로 탑재했습니다. React Native 앱에서 녹음된 사용자의 음성을 로컬 서버에서 텍스트로 즉각 변환하고, LLM이 생성한 텍스트 답변을 다시 자연스러운 음성으로 합성하여 스트리밍합니다. 이를 통해 단순 타이핑이 아닌 실제 '말하기'와 '듣기' 연습이 가능한 완전한 대화형 학습 환경을 구축했습니다.

나의 역할 및 기여

  • Python Flask 기반의 로컬 백엔드 서버 아키텍처 설계 및 구축
  • llama.cpp, Whisper.cpp 등 온디바이스 AI 모델을 연동하고, React Native 클라이언트와 통신할 로컬 API 엔드포인트 구현
  • React Native(TypeScript)를 활용하여 모드(상황극, 자유 대화) 선택 및 실시간 음성/텍스트 입출력(STT/TTS)이 가능한 프런트엔드 UI 개발
  • '상황극' 모드의 AI 페르소나와 대화 흐름을 제어하는 JSON 기반 시나리오 관리자 로직 설계

결과 및 성과

  • 모든 AI 연산을 로컬 PC에서 처리하도록 구현하여, 월간 API 구독 비용 '0원' 및 100% 오프라인 환경에서 동작하는 학습 툴 완성
  • 외부 인터넷으로 데이터를 전송하지 않는 아키텍처를 채택하여, 사용자 대화 내용의 개인정보 유출을 원천적으로 차단
  • 실시간 음성인식(STT)과 음성합성(TTS)을 LLM과 결합하여, '틀릴까 봐 부끄러운' 학습자도 심리적 장벽 없이 연습할 수 있는 몰입형 '말하기/듣기' 환경 제공

회고 (배운 점 및 개선점)

배운 점

  • llama.cpp, Whisper.cpp 등 여러 AI 모델을 로컬 서버 환경에서 통합 구동하고, 이를 Python Flask API로 제어하는 과정을 깊이 있게 경험
  • React Native 앱과 로컬 PC 서버 간의 로컬 네트워크 통신 아키텍처를 직접 설계하고 구현하며, 100% 오프라인 앱의 동작 원리를 이해
  • '상황극 모드'를 구현하며, 시스템 프롬프트 엔지니어링이 AI의 페르소나와 대화 품질에 얼마나 결정적인 영향을 미치는지 실감

개선점

  • 향후 다국어 지원(Multi-language Support)을 추가하고, 각 언어에 최적화된 STT/TTS 및 LLM 모델을 교체할 수 있는 확장성 있는 구조로 개선할 계획
  • 사용자가 문법에 대해 "왜?"라고 질문할 때 더 정확히 답변할 수 있도록, 로컬 RAG(Retrieval-Augmented Generation)를 도입하여 검증된 데이터를 참조해 답변하도록 개선할 것
